# Weather Atlas

A lightweight weather app built with plain HTML, CSS, and JavaScript.

## Features

- City search using Open-Meteo geocoding
- Current conditions
- 24-hour forecast strip
- 7-day forecast
- Celsius as the default temperature unit
- Responsive layout with no build step required

## Run

Open `index.html` in a browser with internet access.

## Notes

The app uses the public Open-Meteo APIs directly from the browser.

## Deploy To GitHub Pages

This project is already set up as a static site for GitHub Pages.

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your main branch and the `/ (root)` folder.
6. Save the settings and wait for GitHub Pages to publish.

Your site URL will be:

`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/`
